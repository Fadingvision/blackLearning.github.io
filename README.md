# 个人博客，基于github和jekyll搭建。
================

####博客地址：https://blackLearning.github.io

####本项目使用了
1. 一些html5新的标签，css3的一些美化效果和过渡效果
2. 自己编写的原生js的日历插件和返回顶部插件
3. 整体页面布局采用Bootstrap构建，有响应式效果，可在移动端访问
4. 多说的网友评论插件

####使用本博客模板
1. fork或download源码
2. 修改相关配置和post文章
3. push到自己的仓库中即可
